Detailed Description

Defines list of actions that corresponds to user commands in sub-systems.

Note
Each sub-system should define a public inherited class(Child) from "ActionList" and also defines his actions as static member functions.
Then in his constructor, all of this actions should be inserted in "Actions" vector base on their "Command IDs - CmdID". For Example:
class SubSysActionList : buplic ActionList
{
public:
SubSysActionList()
{
push_action(&id0Action); // CmdID = 0
// and push other Actions
}
static string id0Action(const XParam::XmlNode *rnode)
throw (Exception)
{
.... may throw Exception
return "result"
}
};
